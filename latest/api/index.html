# LadeGCM.calculate — Method.

calculate(RCP45)

Entry point to the module. Will construct all required parameters and solve the DAE.

Any flags you wish to pass to the solver can be added to the end of the call.

calculate(RCP45, reltol=1e-10, abstol=1e-10, saveat=1765:0.1:2100)

Works as expected.

# LadeGCM.generate_emission_parameters — Method.

(E, LUC) = generate_emission_parameters(RCP45)

For a given concentration pathway, generate continuous functions for fossil fuel emissions E(t) and land use emissions LUC(t).

Data comes from files on disk and is linearly interpolated to provide the continuous output.

source

# LadeGCM.load_pathway_data — Method.

data_frame = load_pathway_data(RCP45)

Loads csv file for a given RCP scenario into a dataframe for processing. The information in these files come from Meinshausen et al. (2011), which were generated using MAGICC6.
